# [59. 螺旋矩阵 II](https://leetcode-cn.com/problems/spiral-matrix-ii/)
## Description
给你一个正整数 `n` ，生成一个包含 `1` 到 `n^2` 所有元素，且元素按顺时针顺序螺旋排列的 `n x n` 正方形矩阵 `matrix` 。
## Example
### Example 1
![](https://assets.leetcode.com/uploads/2020/11/13/spiraln.jpg)  
Input:  
```
n = 3
```
Output:
```
[[1,2,3],[8,9,4],[7,6,5]]
```
### Example 2
Input:  
```
n = 1
```
Output:
```
[[1]]
```
