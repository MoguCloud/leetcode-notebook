# [633. 平方数之和](https://leetcode-cn.com/problems/sum-of-square-numbers/)
## Description
给定一个非负整数 `c` ，你要判断是否存在两个整数 `a` 和 `b`，使得 `a2 + b2 = c` 。
## Example
### Example 1
Input:  
```
c = 5
```
Output:
```
true
```
1 * 1 + 2 * 2 = 5
### Example 2
Input:  
```
c = 4
```
Output:
```
true
```
### Example 3
Input:  
```
c = 3
```
Output:
```
false
```
### Example 4
Input:  
```
c = 2
```
Output:
```
true
```
### Example 5
Input:  
```
c = 1
```
Output:
```
true
```
## Hint
- 0 <= c <= 231 - 1
