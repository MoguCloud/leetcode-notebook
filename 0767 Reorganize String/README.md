# [767. 重构字符串](https://leetcode.cn/problems/reorganize-string/description/)
## Description
给定一个字符串 `s` ，检查是否能重新排布其中的字母，使得两相邻的字符不同。  
返回 `s` 的任意可能的重新排列。若不可行，返回空字符串 `""` 。
## Example
### Example 1
Input:  
```
s = "aab"
```
Output:
```
"aba"
```
### Example 2
Input:  
```
s = "aaab"
```
Output:
```
""
```
## Hint
- 1 <= s.length <= 500
- s 只包含小写字母
