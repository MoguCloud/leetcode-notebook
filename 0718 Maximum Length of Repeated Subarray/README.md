# [718. 最长重复子数组](https://leetcode-cn.com/problems/maximum-length-of-repeated-subarray/)
## Description
给两个整数数组 `A` 和 `B` ，返回两个数组中公共的、长度最长的子数组的长度。
## Example
### Example 1
Input:  
```
A: [1,2,3,2,1]
B: [3,2,1,4,7]
```
Output:
```
3
```
长度最长的公共子数组是 [3, 2, 1] 。
## Hint
- 1 <= len(A), len(B) <= 1000
- 0 <= A[i], B[i] < 100
