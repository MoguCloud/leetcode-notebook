# [1302. 层数最深叶子节点的和](https://leetcode-cn.com/problems/deepest-leaves-sum/)
## Description
给你一棵二叉树，请你返回层数最深的叶子节点的和。  
## Example
### Example 1
![](https://assets.leetcode.com/uploads/2019/07/31/1483_ex1.png)  
Input:  
```
root = [1,2,3,4,5,null,6,7,null,null,null,null,8]
```
Output:
```
15
```
## Hint
- 树中节点数目在 1 到 10^4 之间。
- 每个节点的值在 1 到 100 之间。
