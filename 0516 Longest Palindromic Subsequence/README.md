# [516. 最长回文子序列](https://leetcode-cn.com/problems/longest-palindromic-subsequence/)
## Description
给定一个字符串 `s` ，找到其中最长的回文子序列，并返回该序列的长度。可以假设 `s` 的最大长度为 `1000` 。
## Example
### Example 1
Input:  
```
"bbbab"
```
Output:
```
4
```
一个可能的最长回文子序列为 "bbbb"。
### Example 2
Input:  
```
"cbbd"
```
Output:
```
2
```
一个可能的最长回文子序列为 "bb"。
## Hint
- 1 <= s.length <= 1000
- s 只包含小写英文字母
